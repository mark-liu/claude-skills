# claude-skills-reference

Comprehensive reference for building Claude Code skills — SKILL.md structure, YAML frontmatter spec, progressive disclosure model, testing framework, quality standards, and distribution channels.

## Install

```bash
claude plugin install claude-skills-reference@mark-liu-skills
```

## What This Teaches

| Topic | Coverage |
|-------|----------|
| SKILL.md structure | Required folder layout, YAML frontmatter fields, forbidden patterns |
| Progressive disclosure | 3-level loading model (frontmatter, body, linked files) |
| Use case categories | Document/asset creation, workflow automation, MCP enhancement |
| Key patterns | Sequential orchestration, multi-MCP coordination, iterative refinement, context-aware tool selection, domain intelligence |
| Testing | Manual checks, eval framework, benchmarking, description optimizer |
| Distribution | Claude.ai, Claude Code, API, org-wide deployment |
| Memory integration | Topic files, memory index, backlinks, promotion pipeline |
| skill-creator | Anthropic's official tool — eval framework, A/B comparison, description optimizer |
| Quality standards | Required/recommended sections, common defects from audit of 49 skills |

Pure knowledge — no scripts, no config. Teaches Claude how to build, review, and troubleshoot skills correctly.
